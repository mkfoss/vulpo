Appendix D: CodeBase Limits

The following table lists the limits of CodeBase.

Some CodeBase limits exceed those of xBASE products CodeBase is compatible with (i.e. FoxPro, dBASE, Clipper). Therefore, it is possible to be within CodeBase limits but lose compatibility with other products. Where a CodeBase limit is known to exceed that of another product, the CodeBase limit is listed first.
