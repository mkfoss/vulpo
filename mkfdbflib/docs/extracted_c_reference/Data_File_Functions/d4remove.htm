d4remove

                 Usage:    int d4remove( DATA4 *data ) 

          Description:   This member function permanently removes the data file, its associated index and memo files, from disk.

This function may not be used to delete a data file while a transaction is in progress. Attempts to do so fail and generate an e4transViolation error.

 

This function irrevocably removes the database from disk. If the database contains useful information, consider performing a backup on the database prior to calling d4remove.

 

Before calling d4remove, ensure that the data file has been opened exclusively. Failing to do so can result in CodeBase errors being generated in the user application, or any other application accessing the same data file.

               Returns:

             r4success   The data file and any open index and memo files associated with the data file are deleted from disk.

                       < 0   An error occurred removing the files from disk.
