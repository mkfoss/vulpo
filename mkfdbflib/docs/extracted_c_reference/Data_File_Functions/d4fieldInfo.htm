d4fieldInfo

                 Usage:    FIELD4INFO *d4fieldInfo( DATA4 *data )

          Description:    A pointer to a FIELD4INFO array, which corresponds to the data file, is returned.

This structure is created so it can be used as a parameter to d4create.

The return value becomes obsolete once the data file is closed. This is because the field names of the data file are referenced by the returned FIELD4INFO array.

The FIELD4INFO return value needs to be freed with the function u4free.

               Returns:    A null return means that not enough memory could be allocated.

              See Also:   d4create
