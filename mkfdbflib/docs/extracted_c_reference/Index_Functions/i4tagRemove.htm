i4tagRemove

                 Usage:   int i4tagRemove( TAG4 *tag )

          Description:    Remove a tag from the index. This function is only available for .MDX and .CDX index files.

         Parameters:   

                       tag    The pointer to the TAG4 structure which identifies the tag to be removed. Use d4tag or i4tag to obtain this pointer.

               Returns:   

                         0   Success.

                      < 0   Error.

                  See Also:    d4tag, i4tag
