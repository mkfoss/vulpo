f4dateTime

                 Usage:    char *f4dateTime( const FIELD4 *field )

          Description:   A copy of the field’s contents is returned.

The buffer is overlaid with new data from the field each time f4dateTime is called. See the Warning under f4str for further explanation.

 

 Note

A field in a Visual FoxPro database that is marked as null will still appear to contain its previous value when viewing it. If a field may contain a null value, f4null should first be called.

               Returns:

                Not Null    A pointer to the field’s value, converted into the format CCYYMMDDhh:mm:ss, is returned.

                      Null   An error has occurred.

              See Also:   f4assignDateTime, f4null
