f4null

                 Usage:    int f4null( const FIELD4 *field )

          Description:   This function is used to determine if a field in a Visual FoxPro-compatible table contains a null value.

               Returns:

                         1   The field contains a null value.

                         0   The field does not contain a null value.

                       < 0   An error has occurred.

              See Also:   f4assignNull
