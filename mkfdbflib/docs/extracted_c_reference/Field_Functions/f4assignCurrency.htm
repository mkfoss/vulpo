f4assignCurrency

                 Usage:    void f4assignCurrency( FIELD4 *field, char *data )

          Description:    The parameter data, converted into FoxPro Currency format, replaces the field’s current value.

         Parameters:

                     field    A pointer to the field’s FIELD4 structure.

                     data   A pointer to a null terminated string.

              See Also:   f4assignDouble, f4currency
