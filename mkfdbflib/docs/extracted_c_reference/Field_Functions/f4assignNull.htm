f4assignNull

                 Usage:   void f4assignNull ( FIELD4 *field )

          Description:   The specified field is marked as containing null. This function is only available for the FoxPro file format.

In order for a field to be marked as null, the database must be in the Visual FoxPro file format, and the field must be marked as able to contain null when the database was created. If the database is not in the Visual FoxPro format, an error is generated. If the database is in the Visual FoxPro format, but the field was not marked as able to contain null when the database was created, this function does nothing.

The null marking is removed from the field when any f4assign function or f4blank is called.
