f4assignUnicode

                 Usage:    void f4assignUnicode( FIELD4 *field, const unsigned short *ptr) 

          Description:    The existing field’s value is replaced by the null-terminated Unicode character array pointed to by ptr. If the field is not a Unicode field, an error is generated.

If the length of the new data is less than the field length, the extra space in the field is filled with null characters. If ptr points to too much data, the extra data is ignored.

         Parameters:

                     field    A pointer to the field’s FIELD4 structure.

                        ptr   A pointer to a null-terminated Unicode string.

                  See Also:   f4memoAssign
