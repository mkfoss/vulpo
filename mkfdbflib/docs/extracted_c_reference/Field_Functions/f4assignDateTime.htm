f4assignDateTime

                 Usage:    void f4assignDateTime( FIELD4 *field, char *value )

          Description:    The parameter value, converted into FoxPro DateTime format, replaces the field’s current value.

         Parameters:

                     field    A pointer to the field’s FIELD4 structure.

                    value   A string in the format CCYYMMDDhh:mm:ss. For example, 1:35 PM on December 17, 2001 should be specified as “2001121713:35:00”.

                  See Also:   f4dateTime
