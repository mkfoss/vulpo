f4char

                 Usage:    int f4char ( const FIELD4 *field ) 

          Description:    The first character of the field is returned as an (integer).

 Note

A field in a Visual FoxPro database that is marked as null will still appear to contain its previous value when viewing it. If a field may contain a null value, f4null should first be called.

         Parameters:

                     field    A pointer to the field’s FIELD4 structure.

              See Also:    f4assignChar
