f4true

                 Usage:    int f4true( const FIELD4 *field )

          Description:   This function f4true is used to determine if a logical field is true or false.

 Note

A field in a Visual FoxPro database that is marked as null will still appear to contain its previous value when viewing it. If a field may contain a null value, f4null should first be called.

               Returns:

                         1   The field is true.

                         0   The field is false.

                        <0   An error has occurred.
