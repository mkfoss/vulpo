code4optStart

                 Usage:    int code4optStart( CODE4 *codebase )

          Description:    Initialize CodeBase memory optimization. The amount of memory that is allocated will be less than or equal to the setting of CODE4.memStartMax.

It is not necessary to call code4optStart unless memory optimization is suspended as a result of a call to code4optSuspend. If memory optimization has not yet been started, CodeBase automatically starts memory optimization during any of the following events:

·   When opening a table or file which is eligible for optimization. See CODE4.optimize and CODE4.optimizeWrite for documentation on what makes a file eligible for optimization.

·   When enabling optimization on a table with d4optimize or d4optimizeWrite.

·   When enabling optimization on a file with file4optimize or file4optimizeWrite.

               Returns:

             r4success   Success. Memory optimization has been implemented.

                       < 0   Failure. Memory optimization has not been implemented because there is a lack of available memory or the S4OFF_OPTIMIZE compiler define switch has been defined. This is not considered an error condition since CodeBase may still function without memory optimizations, therefore CODE4.errorCode is not affected.

              See Also:   code4optSuspend, d4optimize, d4optimizeWrite, file4optimize, file4optimizeWrite, CODE4.optimize, CODE4.optimizeWrite, S4OFF_OPTIMIZE

/*ex21.c*/

#include "d4all.h"

extern unsigned _stklen = 10000 ;

 

void main( void )

{

   CODE4 code ;

   DATA4 *dataFile ;

   int delCount, rc ;

 

   code4init( &code ) ;

   code.accessMode = OPEN4DENY_RW ;

 

   dataFile = d4open( &code, "INFO" ) ;

   error4exitTest( &code ) ;

 

   /* initialize optimization with default settings. */

   code4optStart( &code ) ;

 

   delCount = 0 ;

   for( rc = d4top( dataFile ); rc == r4success ; rc = d4skip( dataFile, 1L ) )

       if( d4deleted( dataFile) )

          delCount++ ;

 

   printf("%d records are marked for deletion.\n", delCount ) ;

   code4initUndo( &code ) ;

}
