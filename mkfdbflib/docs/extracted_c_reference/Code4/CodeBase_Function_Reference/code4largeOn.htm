code4largeOn

                 Usage:   void code4largeOn( CODE4 *codebase )

          Description:   This function will turn on the large file support. Once it has been turned on it can not be turned off without restarting the application. This function must be used after initialization and before any data files are opened.

In applications that use multiple CODE4 structures, the first CODE4 initialized must use this function before initializing another CODE4 structure or an error will occur.

This function causes CodeBase to use a different locking algorithm that is not compatible with xBASE applications. Ensure that all applications that are accessing the same files are using the same locking algorithm. If multiple applications share the same files but use different locking algorithms, the database files can become corrupt.
