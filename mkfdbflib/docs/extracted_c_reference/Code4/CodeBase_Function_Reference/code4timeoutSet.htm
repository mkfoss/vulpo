code4timeoutSet

                 Usage:    void code4timeoutSet( CODE4 *codebase, long setting )

          Description:    This function sets the value of the CODE4 member variable CODE4.timeout, which is used by CodeBase to determine how long it should wait to receive a message from the server, before disconnecting. CODE4.timeout measures time in seconds. A value of WAIT4EVER indicates that the application will wait forever for a response from the server.

The default value is WAIT4EVER if the S4TIMEOUT_HOOK switch is not defined. If the switch is defined, then the default value is 100.

If the S4TIMEOUT_HOOK switch is not defined, then CodeBase will generate an error when it times out. If the switch is defined, CodeBase will call the time out hook function (code4timeoutHook) when it times out.

              See Also:   code4timeoutHook, S4TIMEOUT_HOOK, code4timeout
