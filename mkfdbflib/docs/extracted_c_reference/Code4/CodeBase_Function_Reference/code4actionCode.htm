code4actionCode

                 Usage:    unsigned short code4actionCode( CODE4 *codebase )

          Description:    This function can be used together with d4reindexWithProgress to determine if CodeBase is busy processing an action.

               Returns:   

          ACTION4NONE    CodeBase is idle.

ACTION4REINDEXING    CodeBase is busy reindexing a table.

ACTION4INITIALIZING    CodeBase is preparing to begin a requested action.

              See Also:   d4reindexWithProgress
