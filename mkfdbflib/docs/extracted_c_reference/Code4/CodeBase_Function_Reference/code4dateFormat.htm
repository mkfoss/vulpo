code4dateFormat

                 Usage:   const char *code4dateFormat( CODE4 *codeBase )

          Description:   This function returns the current default date format for the client application. This is equivalent to accessing the CODE4.date_format member in CodeBase 5.1. The initial date format is "MM/DD/YY". The date format is used when an expression involving the dBASE functions CTOD() or DTOC() are involved.
