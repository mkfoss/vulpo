CODE4.useGeneralTagsInRelate

                 Usage:    short CODE4.useGeneralTagsInRelate = 0

          Description:    This true/false flag determines whether the CodeBase Relation/Query module can use that use General collation to enable Query Optimization. If CODE4.useGeneralTagsInRelate is true (non-zero), any tags can be used to enable Query Optimization. This behavior can also be achieved by defining the S4USE_GENERAL_TAGS_IN_RELATE switch and rebuilding the CodeBase library. If CODE4.useGeneralTagsInRelate is false (zero), only tags that use Machine collation will be used to enable Query Optimization.

The default setting is false (zero).

              See Also:   code4collate, relate4querySet, relate4optimizeable
