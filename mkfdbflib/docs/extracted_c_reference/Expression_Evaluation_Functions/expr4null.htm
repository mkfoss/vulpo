expr4null

                 Usage:    int expr4null( EXPR4 *expr )

          Description:    The expression is evaluated to determine if the result is null.

         Parameters:

                     expr   A pointer to the expression’s EXPR4 structure.

               Returns:

                       > 0    The expression evaluates to null.

                         0    The expression does not evaluate to null.

                       < 0    Error.
